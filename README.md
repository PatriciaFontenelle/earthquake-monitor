# Earthquake Monitor

Earthquake monitor is a Web Application that lists the latest earthquakes.

## SETUP
After cloning the project:
- Run ```npm i ```
- Run ```npm run dev```
- Check the terminal to see in which port the project is running and open it on you browser using localhost.

## Technologies
Here are the technologies used to develop this application:
- ReactJS
- Vite
- Tailwind
